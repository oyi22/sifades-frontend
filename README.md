# SIFADES - Frontend

Frontend aplikasi **Absensi Face Recognition untuk Perangkat Desa**  
dibangun menggunakan React + Vite + TypeScript.

---

##  Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router DOM
- TanStack React Query
- Axios
- Zustand
- React Hook Form + Zod
- Recharts
- Lucide React
- date-fns
- clsx + tailwind-merge
- sonner (toast)
- radix-ui

---

##  UI & Styling

- Tailwind CSS
- shadcn/ui component system
- tw-animate-css
- class-variance-authority (CVA)
